## Getting Started

### Prerequisites
- Node.js (v14 or higher)

### Installation
```bash
npm install
```

### Running the App
```bash
npm run dev
```

## Project Structure
- `src/`: Main application source code (React).
- `public/`: Static assets (icons).
- `legacy_extension/`: Backup of the original Chrome Extension files.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Pure CSS (CSS Variables, Flexbox, Glassmorphism)
- **APIs**: Web Speech API, Google Translate API
